/// 01

O Método

Visão geral

Este ebook ensina um fluxo de trabalho com quatro ferramentas que, juntas, permitem criar e publicar uma landing page profissional em menos de 15 minutos — mesmo sem qualquer conhecimento técnico.

Regra de ouro: landing page que converte é linear. Sem menus complexos, sem múltiplas páginas. O usuário rola para baixo e converte.

01
Escolha o template base no Aura Build ↗

Selecione um modelo com Hero, Prova, Método, Depoimentos, Oferta e CTA. Evite templates com menus complexos ou múltiplas páginas — landing page que converte é linear.

02
Crie o prompt visual no ChatGPT ↗

Defina paleta de cores, tipografia, estilo visual e estética da marca. O ChatGPT devolve um briefing completo pronto para instruir o Claude.

03
Gere a página completa no Claude ↗

Cole o script base + prompt visual e receba um HTML pronto para publicação — otimizado, responsivo, com todos os CTAs configurados.

04
Publique no GitHub Pages ↗ — de graça

Suba o arquivo index.html e seu site fica online com link próprio, sem custo. Nenhum servidor, nenhuma assinatura, zero reais.

/// 04

Os Prompts
Prontos

Copie e use

O guia inclui todos os prompts prontos para copiar e colar. Abaixo estão os três principais — do briefing visual à configuração final dos CTAs.

Prompt 01 — ChatGPT · Briefing Visual

Use este prompt no ChatGPT ↗ para gerar o briefing visual completo da sua página. Copie o resultado e use no Prompt 02:

Crie um prompt de design visual para uma landing page.

Nicho: [seu nicho]
Produto: [seu produto]
Público: [quem compra]

Quero definir:
- paleta de cores
- tipografia
- estilo visual
- estética da marca
- estilo das seções
- estilo dos botões
- estilo de ícones
- estilo das imagens

A estética deve ser moderna, premium e focada em conversão.

O ChatGPT vai devolver um bloco com cores, fontes, layout e experiência visual. Copie o resultado completo — ele será colado no Prompt 02 como identidade visual da página.

Prompt 02 — Claude AI · Gerar a Página

Cole este prompt no Claude ↗ com o script do Aura Build e o briefing do ChatGPT:

Vou criar uma landing page.

Aqui está o script base:
[cole o script do Aura Build]

Agora quero que você:

1. Reestruture para uma landing page de alta conversão
2. Aplique a estética visual abaixo:
  [cole o resultado do ChatGPT]
3. Otimize para: carregamento rápido,
  design moderno, mobile first, seções organizadas
4. Inclua: Hero · Benefícios · Método ·
  Depoimentos · Oferta · CTA final
5. Gere um arquivo completo em HTML + CSS

Nome do arquivo: index.html
Prompt 03 — Claude AI · Configurar CTAs

Após gerar a página, substitua todos os botões de venda com um único comando:

Coloque este link em todos os botões de CTA da página:
https://pay.hotmart.com/seuproduto

Todos os botões devem abrir o link em nova aba.

Funciona com WhatsApp também: substitua pelo link wa.me/55SEU_NUMERO?text=... e o Claude atualiza todos os botões automaticamente.

/// 05

Publicar
de Graça

GitHub Pages

O GitHub Pages ↗ hospeda sites estáticos completamente de graça. Em menos de 5 minutos, seu index.html fica online com link público e profissional.

Criar conta em github.com ↗ Gratuito, sem cartão. Clique em Sign Up e siga o processo de cadastro.
Criar um repositório público New Repository → nomeie o projeto → marque Public → Create repository
Fazer upload do index.html Add File → Upload Files → arraste o arquivo index.html → Commit changes
Ativar o GitHub Pages Settings → Pages → Source: Deploy from branch → Branch: main → /root → Save
Seu site está no ar Link: seuusuario.github.io/landing-page-produto/
/// 06

Testar Antes
de Lançar

QA checklist

Antes de divulgar, teste tudo. Se algo quebrar, o Claude corrige com um único comando de texto — sem precisar entender uma linha de código.

Mobile

Layout responsivo, botões clicáveis, texto legível sem zoom

Desktop

Seções alinhadas, grid correto, imagens carregando

Botões CTA

Links corretos, abrindo em nova aba, todos funcionando

Velocidade

Carregamento rápido, sem erros no console

Se algo quebrar, envie ao Claude: "No mobile o botão CTA está desalinhado. Corrija." — Retorna o código corrigido. Substitua o arquivo no GitHub e pronto.
